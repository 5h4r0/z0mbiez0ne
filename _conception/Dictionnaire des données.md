# 📖 Dictionnaire des Données (PostgreSQL)

SGBD = PostgreSQL

## Pour toutes les tables sauf Role et Order :

- `GENERATED` = `INTEGER GENERATED BY DEFAULT AS IDENTITY` 
- Ajout des champs : 
  - `created_at` : `TYPESTAMPTZ DEFAULT NOW()`
  - `updated_at` : `TYPESTAMPTZ DEFAULT NOW()`
  - `deleted_at` : `TIMESTAMPTZ DEFAULT NULL`

Notes : 
- Autre option pour l'`id` : `SERIAL` (spécifique à Postgres, pas dans la norme SQL)
- Autre option pour `firstname` : `VARCHAR(50)` (limiter la taille du texte)

---

## **user**
| Colonne       | Type        | Contraintes                        | Description                |
|---------------|-------------|------------------------------------|----------------------------|
| user_id       | SERIAL (PK) | PRIMARY KEY                        | Identifiant unique du user |
| role_id       | INT (FK)    | REFERENCES role(role_id), NOT NULL | Rôle attribué              |
| email         | VARCHAR(50) | UNIQUE, NOT NULL                   | Adresse email              |
| firstname     | VARCHAR(50) | NOT NULL                           | Prénom                     |
| lastname      | VARCHAR(50) | NOT NULL                           | Nom                        |
| password_hash | CHAR(60)    | NOT NULL                           | Mot de passe hashé         |
| created_at    | TIMESTAMP   | DEFAULT now(), NOT NULL            | Date de création           |
| updated_at    | TIMESTAMP   | DEFAULT now(), NOT NULL            | Dernière modification      |
| deleted_at    | TIMESTAMP   | NULL                               | Soft delete (NULL = actif) |

## **role**
| Colonne   | Type        | Contraintes                         | Description |
|-----------|-------------|-------------------------------------|-------------|
| role_id   | SERIAL (PK) | PRIMARY KEY                         | Identifiant du rôle |
| name      | VARCHAR(50) | CHECK (name IN ('member','admin'))  | Nom du rôle |

## **activity**

| deleted_at  | TIMESTAMP    | NULL                                       | Soft delete |

## **activity_session**

| activity_id  | INT (FK)     | REFERENCES activity(activity_id), NOT NULL                            | Activité associée |

| updated_at   | TIMESTAMP    | DEFAULT now(), NOT NULL                                               | Dernière modification |
| deleted_at   | TIMESTAMP    | NULL                                                                  | Soft delete |

## **cart**

|---------------|--------------|------------------------------------------------------------------------|-------------|
| cart_id       | SERIAL (PK)  | PRIMARY KEY                                                            | Identifiant unique |
| user_id       | INT (FK)     | REFERENCES "user"(user_id), NOT NULL                                   | Propriétaire du panier |
| total_price   | NUMERIC(10,2)| NOT NULL                                                               | Montant total |

| payment_date  | TIMESTAMP    | NULL                                                                  | Date de paiement |

| updated_at    | TIMESTAMP    | DEFAULT now(), NOT NULL                                               | Dernière modification |

## **order**
| Colonne     | Type         | Contraintes                                                              | Description |
|-------------|--------------|--------------------------------------------------------------------------|-------------|
| order_id    | SERIAL (PK)  | PRIMARY KEY                                                              | Identifiant unique |
| cart_id     | INT (FK)     | REFERENCES cart(cart_id), NOT NULL                                       | Panier associé |
| session_id  | INT (FK)     | REFERENCES activity_session(session_id), NOT NULL                        | Session réservée |
| tickets_qty | INT          | NOT NULL, CHECK (tickets_qty > 0)                                        | Quantité de tickets |
| unit_price  | NUMERIC(10,2)| NOT NULL                                                                 | Prix unitaire |
| created_at  | TIMESTAMP    | DEFAULT now(), NOT NULL                                                  | Date de création |
| updated_at  | TIMESTAMP    | DEFAULT now(), NOT NULL                                                  | Dernière modification |
| UNIQUE      | (cart_id, session_id) | Empêche plusieurs lignes pour une même session dans un panier   | Contrainte d’unicité |
