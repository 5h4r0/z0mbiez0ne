# 📖 Dictionnaire des Données

*Ajout des champs avec TZ si Time Zone : TYPESTAMPTZ DEFAULT NOW()
Limiter la taille du texte : `firstname` : `VARCHAR(50)`*

## SGBD = PostgreSQL

---

## **roles**
| Colonne | Type        | Contraintes                          | Description         |
|---------|-------------|--------------------------------------|---------------------|
| id      | INTEGER     | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant du rôle |
| name    | VARCHAR(20) | CHECK (name IN ('member','admin'))   | Nom du rôle         |

---

## **users**
| Colonne       | Type         | Contraintes                          | Description           |
|---------------|--------------|--------------------------------------|-----------------------|
| id            | INTEGER      | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant unique    |
| roles_id      | INTEGER      | FK → role(id), NOT NULL              | Rôle attribué         |
| email         | VARCHAR(320) | UNIQUE, NOT NULL                     | Adresse email         |
| firstname     | VARCHAR(50)  | NOT NULL                             | Prénom                |
| lastname      | VARCHAR(50)  | NOT NULL                             | Nom                   |
| password_hash | VARCHAR(100) | NOT NULL                             | Hash du mot de passe  |
| created_at    | TIMESTAMP    | DEFAULT now(), NOT NULL              | Date de création      |
| updated_at    | TIMESTAMP    | DEFAULT now(), NOT NULL              | Dernière modification |
| deleted_at    | TIMESTAMP    | NULL                                 | Soft delete           |

---

## **categories**
| Colonne     | Type          | Contraintes                          | Description           |
|-------------|---------------|--------------------------------------|-----------------------|
| id          | INTEGER       | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant           |
| title       | VARCHAR(100)  | NOT NULL                             | Nom                   |
| description | VARCHAR(2000) | NULL autorisé                        | Description           |
| image       | VARCHAR(50)   | NULL autorisé                        | URL image             |
| created_at  | TIMESTAMP     | DEFAULT now(), NOT NULL              | Date de création      |
| updated_at  | TIMESTAMP     | DEFAULT now(), NOT NULL              | Dernière modification |
| deleted_at  | TIMESTAMP     | NULL                                 | Soft delete           |

---

## **activities**
| Colonne     | Type          | Contraintes                          | Description           |
|-------------|---------------|--------------------------------------|-----------------------|
| id          | INTEGER       | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant           |
| title       | VARCHAR(255)  | NOT NULL                             | Nom                   |
| description | VARCHAR(5000) | NULL autorisé                        | Description           |
| image       | VARCHAR(50)   | NULL autorisé                        | URL image             |
| created_at  | TIMESTAMP     | DEFAULT now(), NOT NULL              | Date de création      |
| updated_at  | TIMESTAMP     | DEFAULT now(), NOT NULL              | Dernière modification |
| deleted_at  | TIMESTAMP     | NULL                                 | Soft delete           |

---

## **activities_categories**
| Colonne       | Type    | Contraintes                   | Description        |
|---------------|---------|-------------------------------|--------------------|
| activities_id | INTEGER | FK → activities(id), NOT NULL | Activité associée  |
| categories_id | INTEGER | FK → categories(id), NOT NULL | Catégorie associée |

---

## **sessions**
| Colonne       | Type         | Contraintes                                                       | Description           |
|---------------|--------------|-------------------------------------------------------------------|-----------------------|
| id            | INTEGER      | GENERATED BY DEFAULT AS IDENTITY, PK                              | Identifiant           |
| activities_id | INTEGER      | FK → activities(id), NOT NULL                                     | Activité associée     |
| date          | TIMESTAMP    | NOT NULL                                                          | Date/heure            |
| capacity      | INTEGER      | NOT NULL                                                          | Capacité totale       |
| unit_price    | NUMERIC(3,2) | NOT NULL                                                          | Prix                  |
| status        | VARCHAR(10)  | CHECK (status IN ('Scheduled','Cancelled','Completed')), NOT NULL | Statut                |
| created_at    | TIMESTAMP    | DEFAULT now(), NOT NULL                                           | Date de création      |
| updated_at    | TIMESTAMP    | DEFAULT now(), NOT NULL                                           | Dernière modification |
| deleted_at    | TIMESTAMP    | NULL                                                              | Soft delete           |

---

## **orders_lines**
| Colonne         | Type                             | Contraintes                                                     | Description                |
|-----------------|----------------------------------|-----------------------------------------------------------------|----------------------------|
| id              | INTEGER                          | GENERATED BY DEFAULT AS IDENTITY, PK                            | Identifiant                |
| orders_id       | INTEGER                          | FK → orders(id), NOT NULL                                       | Commande associée          |
| sessions_id     | INTEGER                          | FK → sessions(id), NOT NULL                                     | Session réservée associées |
| tickets_qty     | INTEGER                          | NOT NULL, CHECK (tickets_qty > 0)                               | Nombre de tickets          |
| Amount          | NUMERIC(3,2)                     | NOT NULL                                                        | Prix unitaire              |
| created_at      | TIMESTAMP                        | DEFAULT now(), NOT NULL                                         | Date de création           |
| updated_at      | TIMESTAMP                        | DEFAULT now(), NOT NULL                                         | Dernière modification      |
| orders_sessions | (orders_id, sessions_id), UNIQUE | Empêche plusieurs lignes pour la même session dans une commande | Contrainte                 |

---

## **orders**
| Colonne        | Type         | Contraintes                                                                | Description           |
|----------------|--------------|----------------------------------------------------------------------------|-----------------------|
| id             | INTEGER      | GENERATED BY DEFAULT AS IDENTITY, PK                                       | Identifiant           |
| users_id       | INTEGER      | FK → users(id), NOT NULL                                                   | Propriétaire          |
| taxes          | NUMERIC(3,2) | DEFAULT 0, NOT NULL                                                        | Taxes                 |
| total_amount   | NUMERIC(4,2) | NULL                                                                       | Montant total         |
| payment_method | VARCHAR(30)  | NULL                                                                       | Méthode de paiement   |
| payment_date   | TIMESTAMP    | NULL                                                                       | Date de paiement      |
| status         | VARCHAR(10)  | CHECK (status IN ('Pending','Confirmed','Cancelled','Refunded')), NOT NULL | Statut                |
| created_at     | TIMESTAMP    | DEFAULT now(), NOT NULL                                                    | Date de création      |
| updated_at     | TIMESTAMP    | DEFAULT now(), NOT NULL                                                    | Dernière modification |
| deleted_at     | TIMESTAMP    | NULL                                                                       | Soft delete           |
