# 📖 Dictionnaire des Données

*Source de vérité : `backend/src/models/schema.prisma`*

## SGBD = PostgreSQL

---

## **roles**
| Colonne | Type        | Contraintes                          | Description         |
|---------|-------------|--------------------------------------|---------------------|
| id      | INTEGER     | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant du rôle |
| name    | VARCHAR(20) | UNIQUE, NOT NULL                     | Nom du rôle         |

---

## **users**
| Colonne       | Type         | Contraintes                          | Description           |
|---------------|--------------|--------------------------------------|-----------------------|
| id            | INTEGER      | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant unique    |
| role_id       | INTEGER      | FK → roles(id), NOT NULL             | Rôle attribué         |
| email         | VARCHAR(320) | UNIQUE, NOT NULL                     | Adresse email         |
| firstname     | VARCHAR(50)  | NOT NULL                             | Prénom                |
| lastname      | VARCHAR(50)  | NOT NULL                             | Nom                   |
| password_hash | VARCHAR(150) | NOT NULL                             | Hash argon2           |
| created_at    | TIMESTAMP    | DEFAULT now(), NOT NULL              | Date de création      |
| updated_at    | TIMESTAMP    | DEFAULT now(), NOT NULL              | Dernière modification |
| deleted_at    | TIMESTAMP    | NULL                                 | Soft delete           |

## **activities_categories**
| Colonne     | Type    | Contraintes                              | Description        |
|-------------|---------|------------------------------------------|--------------------|
| activity_id | INTEGER | FK → activities(id), NOT NULL, PK (comp) | Activité associée  |
| category_id | INTEGER | FK → categories(id), NOT NULL, PK (comp) | Catégorie associée |

*PK composite : `(activity_id, category_id)`*

## **sessions**
| Colonne     | Type         | Contraintes                          | Description           |
|-------------|--------------|--------------------------------------|-----------------------|
| id          | INTEGER      | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant           |
| activity_id | INTEGER      | FK → activities(id), NOT NULL        | Activité associée     |

| capacity    | INTEGER      | NOT NULL                             | Capacité totale       |
| unit_price  | DECIMAL(5,2) | NOT NULL                             | Prix unitaire         |
| status      | SessionStatus| NOT NULL                             | Statut                |
| created_at  | TIMESTAMP    | DEFAULT now(), NOT NULL              | Date de création      |
| updated_at  | TIMESTAMP    | DEFAULT now(), NOT NULL              | Dernière modification |
| deleted_at  | TIMESTAMP    | NULL                                 | Soft delete           |

## **orders**
| Colonne        | Type         | Contraintes                          | Description           |
|----------------|--------------|--------------------------------------|-----------------------|
| id             | INTEGER      | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant           |
| user_id        | INTEGER      | FK → users(id), NOT NULL             | Propriétaire          |

| updated_at     | TIMESTAMP    | DEFAULT now(), NOT NULL              | Dernière modification |

## **orders_lines**
| Colonne     | Type          | Contraintes                          | Description                |
|-------------|---------------|--------------------------------------|----------------------------|
| id          | INTEGER       | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant                |
| order_id    | INTEGER       | FK → orders(id), NOT NULL            | Commande associée          |
| session_id  | INTEGER       | FK → sessions(id), NOT NULL          | Session associée           |
| tickets_qty | INTEGER       | NOT NULL                             | Nombre de billets          |
| amount      | DECIMAL(10,2) | NOT NULL                             | Montant de la ligne        |
| created_at  | TIMESTAMP     | DEFAULT now(), NOT NULL              | Date de création           |
| updated_at  | TIMESTAMP     | DEFAULT now(), NOT NULL              | Dernière modification      |

*Contrainte UNIQUE : `(order_id, session_id)` — une seule ligne par session par commande*

## **refresh_token**
| Colonne    | Type        | Contraintes                          | Description                    |
|------------|-------------|--------------------------------------|--------------------------------|
| id         | INTEGER     | GENERATED BY DEFAULT AS IDENTITY, PK | Identifiant                    |
| token_id   | VARCHAR(64) | UNIQUE, NOT NULL                     | UUID stocké dans le cookie JWT |
| token_hash | TEXT        | NOT NULL                             | Hash argon2 du token_id        |
| user_id    | INTEGER     | FK → users(id), NOT NULL             | Utilisateur propriétaire       |
| issued_at  | TIMESTAMP   | DEFAULT now(), NOT NULL              | Date d'émission                |
| expired_at | TIMESTAMP   | NOT NULL                             | Date d'expiration              |
